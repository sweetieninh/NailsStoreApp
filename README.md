# NailsStoreApp

Expo React Native TypeScript app for the nail salon customer check-in flow.

## Features
- Check-in phone lookup screen with large keypad
- Auto-formatted phone input
- Welcome screen for returning customers
- Registration screen for new customers
- API calls to the NailsRestServer backend at http://192.168.9.152:4010/api

## Run with Expo Go
1. Install dependencies:
   npm install
2. Start Expo:
   npm start
3. Open the Expo Go app on your phone and scan the QR code.
4. Make sure the backend API is running at http://192.168.9.152:4010
